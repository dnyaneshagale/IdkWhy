IdkWhy
Temporary clipboard bridge
No login · No database · In-memory only
Fast handoff between devices

Copy text on your phone. Paste it on your PC.

IdkWhy keeps the workflow intentionally small: create a room, join it from another device, and move text across without chat history, accounts, or external storage.

50 Maximum messages per room, newest first.
2d Rooms expire after 2 days of inactivity and are cleaned up automatically.
2s Room messages refresh automatically without reloading the page.
UUID Every message gets a unique ID for copy and delete actions.

Create Room

Choose a 6-12 character room number. Optionally add a password for private access.

Join Room

Open an existing room from another device and continue sharing the clipboard payload.

How it works

Create a room on the sending device, open the same room on the receiving device, and use the message cards as a short-lived clipboard queue.
